Color Scheme
Primary: Crimson red / blood red (#DC2626 / #B91C1C) — bold, aggressive, on-brand for TachikomaRed
Accent: Near-black dark backgrounds (#0A0A0A, #111111) — makes red pop, feels techy
Secondary accent: Cold steel gray (#374151, #6B7280) — mecha / robotic feel
Text: Off-white / light gray on dark (#F9FAFB, #D1D5DB)
Highlight: Bright red glow for CTAs and rewards
Visual Style
Vibe: Minimalist meme — clean layout but with sarcastic, irreverent personality baked in
Aesthetic: Dark mode first, mecha/robotic edge, subtle glitch/tech vibes
Typography feel: Sharp, tight, slightly aggressive — no rounded softness
Tone: Dry humor, sarcastic AI agent energy — like TachikomaRed would roll its eyes at you but still help
NOT: Cute, pastel, playful — this is a battle-hardened crab agent
Additional Notes
Inspired by TachikomaRed — a sarcastic mecha crab AI agent (bankr agent) that builds and deploys to improve life for agents and humans
Think Ghost in the Shell meets degen meme culture
Red is the brand color — lean into it heavily (borders, glows, accents)
Minimalistic layout with maximum personality in the copy/iconography
Dark backgrounds mandatory — light mode would be a war crime for this brand
Crab/mecha imagery welcome in assets
